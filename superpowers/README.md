# Superpowers

This directory is for non-shipping planning material that helps steer the product and repository.

Use `superpowers/` for material such as:

- product strategy notes
- roadmap and backlog tracking
- positioning and refinement plans
- docs information architecture decisions
- growth and go-to-market planning

Do not use this directory for:

- maintainer implementation docs that should ship with the repository
- public user documentation that must stay versioned with code paths
- code-local references that belong next to the implementation

Directory guide:

- [`roadmap.md`](/users/Yuyang/Others/DailyReport/superpowers/roadmap.md) — tracked roadmap and technical debt
- [`refinement-todo.md`](/users/Yuyang/Others/DailyReport/superpowers/refinement-todo.md) — current product/page refinement plan
- [`wiki-strategy.md`](/users/Yuyang/Others/DailyReport/superpowers/wiki-strategy.md) — recommendation for GitHub Wiki vs in-repo docs
